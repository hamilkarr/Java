입출력(입력,출력)
IO -> Input,Output
스트림 -> 데이터가 이동하는 가상의 통로

단방향 스트림
  - 입력 스트림
  - 출력 스트림

입력스트림
  - InputStream : 추상 클래스  (바이트 단위의 입력 스트림)
      -> 바이트 단위의 입력스트림이 따라야할 규칙, 규약을 정의하는 추상 클래스
      FileInputStream
      BufferedInputStream

  - Reader : 추상 클래스  (문자 단위의 입력 스트림)
      -> 문자 단위의 입력 스트림이 따라야할 규칙, 규약을 정의하는 추상 클래스
      FileReader
      FilterInputStream
      BufferedReader
      DataInputStream
      ObjectInputStream
  
출력스트림
  - OutputStream : 추상 클래스 ( 바이트 단위의 출력 스트림)
      -> 바이트 단위의 출력 스트림이 따라야할 규칙,규약을 정의한 추상 클래스
      FileOutputStream
      BufferedOutputStream

  - Writer : 추상 클래스 ( 문자 단위의 출력 스트림)
      -> 문자 단위의 출력 스트림이 따라야할 규칙, 규약을 정의한 추상 클래스
      FileWriter
      BufferdWriter

기반스트림
  - 직접 읽고 쓰는 스트림

보조스트림
  - 직접 읽고 쓰는 기능은 없음
  - 기반 스트림에  기능을 추가함
  - 기반스트림 + 보조스트림 + 보조스트림 (추가기능은 여러개 추가 가능)

  BufferedInputStream
  BufferedOutputStream

  BufferedReader
  BufferedWriter
