프로그래밍 언어 중 하나인 자바를 사용하여 하기 프로그램을 팀별로 2개씩 선택하여 제작하시오.

  1. 학점 관리 프로그램
  2. 도서 관리 프로그램 
  3. 가계부 관리 프로그램
  4. 주소록 관리 프로그램

유의사항
  1) 각 팀에서 2,3명씩 분할하여 2개의 주제를 정하시오
  2) 프로그램의 제작은 조별 스터디를 통해서 충분히 학습 후 제작하시오
  3) 학점 관리 프로그램은 교재에 예시로 나와 있지만 교재와는 다르게 제작하셔야 합니다.
     (소스의 유사성이 30% 이상일 경우 0점 처리)
  4) 데이터의 등록, 조회, 변경을 할 수 있도록 제작하여야 합니다.
  5) 4번의 항목은 콘솔 프로그램의 형태로 메뉴가 구성이 되고 메뉴를 선택하면 해당 기능으로 
     전환되도록 구성되어야 합니다.
