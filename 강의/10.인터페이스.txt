  모니터 
    - 액정 (LCD,LED) - 인치수, LCD, LED, 밝기
    - 메인보드 (제품군에 따른 규격)
    - 파워서플라이 (220V, 110V)
    - 모니터와 본체를 연결하는 IO 포트
      (RGB, HDMI, USB-C, DVI, DP)

  모니터의 구성 요소의 각 표준 정의

  interFace : 하나의 클래스에서 여러개의 인터페이스를 구현
  class 대신 interface 예약어
  implements -> 인터페이스 구현

  각 장치별 표준, 규약 -> 추상 메서드
    -> abstract 예약어는 생략 -> 컴파일러에서 자동 추가 public abstract

  메서드의 구현 부분이 없으므로 인스턴스 생성 불가 X
  하위 클래스에서 구현을 통해서
    -> 인스턴스 생성이 불가 -> 멤버 변수 X
    -> 인터페이스내에 정의되어 있는 값 -> static 상수
