08-1) 상속이란
  상위 클래스에서 하위 클래스로 상속 -> 하위 클래스에 있는 멤버 변수, 메서드를 사용 가능

  클래스 상속 문법
    extends

  하위 클래스 extends 상위클래스 {

  }

    B extends A

    a: numA, getNumA()
    b: numA, getNumA() + numB, getNumB()

  상속을 해준 클래스(부모클래스) A -> 일반적이고
  상속을 받은 클래스(자식클래스) B -> 부모 클래스 보다 구체적이다

  ex) 회원등급
  실버 -> 일반회원
  VIP -> 일반회원 혜택 + 추가적인 부가혜택
